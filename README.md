# Геодезические калькуляторы

Два полевых инструмента в одном компактном HTML-файле. Работает на телефоне, планшете, ноутбуке — без интернета, без установки. Открыл браузер и считаешь.

---

## 📐 Калькулятор №1. Смещение точки вдоль базиса

**Задача:**  
Есть отрезок A–B (базис). Нужно отложить от точки B заданное расстояние (2, 3, 5 м) в том же направлении A→B и получить координаты новой точки C.

**Входные данные:**
- X, Y точки A (начало базиса)
- X, Y точки B (конец базиса, от него откладываем)
- величина смещения в метрах

**Выход:**
- X, Y точки C с точностью 4 знака после запятой

**Формула:**  
Единичный вектор направления A→B умножается на смещение и прибавляется к точке B.

**Когда нужно:**  
– вынос проектной точки по створу  
– разбивка осей, фасадов, линий связи  
– контроль отступов от красных линий

## 🧮 Калькулятор №2. Теорема Пифагора

**Задача:**  
По двум катетам прямоугольного треугольника вычислить гипотенузу.

**Формула:**  
c = √(a² + b²)

**Когда нужно:**  
– проверка диагоналей  
– восстановление перпендикуляров  
– расчёт длины откоса / фермы / раскоса  
– любые «косые» размеры, где нужен прямой угол

---

## ⚙️ Как пользоваться

1. Скачай файл `index.html` (или открой по ссылке GitHub Pages)
2. Работает без интернета
3. Переключай вкладки вверху карточки
4. Вводи цифры — результат появляется сразу
